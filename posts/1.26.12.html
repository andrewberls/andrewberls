This doesn't look like much of anything, so let's add some style to our button class.

# style.scss
.button {
	background-color: #3099DF;	  
	color: white;  
	font-family: Arial, Verdana, Geneva, sans-serif;
	font-size: 0.8em;     
	padding: 7px;
}

That's a good start - all we're doing here is some font basics, adding a nice blue background color, and some padding to make it look more like a button.
